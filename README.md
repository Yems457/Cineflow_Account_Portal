# Cineflow Account Portal

## Project Description
The CineFlow Account Portal is a secure, user-facing application allowing users to manage their profiles, video uploading, channels preferences, and viewing history.

## Key Features
* **User Authentication:** Personal Information, Username, Password, Register Account and Clear Form

### Installation
  Clone the repository
   ```sh
   git clone https://github.com/Yems457/Cineflow_Account_Portal.git
   ```

## Usage

Open the local development URL shown by the command and sign in to manage accounts.
